# 8-Puzzle Solver

This project implements several classical Artificial Intelligence search algorithms to solve the 8‑Puzzle problem and compare their performance.

## Algorithms
- Breadth-First Search (BFS)
- Depth-First Search (DFS)
- Iterative Deepening Search (IDS)
- Uniform Cost Search (UCS)
- A* Search  
  - Misplaced Tiles heuristic  
  - Manhattan Distance heuristic  

## Usage

Clone the repository:

git clone https://github.com/fereshtehghaderi2001-glitch/ai-search-algorithms-8puzzle.git

Run the program:

python main.py
